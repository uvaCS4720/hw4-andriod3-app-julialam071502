# UVA Campus Map

An Android app that displays UVA campus locations on a Google Map.

## Features
- Google Map showing UVA campus locations
- Filter locations by tag using dropdown menu
- Defaults to "core" tag on startup
- Tap markers to see building name and description
- Data cached in SQLite database (no duplicates on re-run)
